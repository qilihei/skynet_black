# 业务模块目录

这个目录包含心桥物语游戏的各个业务模块。

## 模块列表

### 核心模块
- `player/` - 玩家系统，管理玩家基础信息
- `farm/` - 农场系统，种植、收获、动物养殖
- `inventory/` - 背包系统，物品管理
- `shop/` - 商店系统，买卖交易

### 社交模块  
- `friend/` - 好友系统
- `guild/` - 公会系统
- `chat/` - 聊天系统
- `mail/` - 邮件系统

### 任务模块
- `quest/` - 任务系统
- `achievement/` - 成就系统
- `daily/` - 日常任务

### 经济模块
- `market/` - 市场交易
- `auction/` - 拍卖行
- `bank/` - 银行系统

### 活动模块
- `event/` - 活动系统
- `festival/` - 节日活动
- `competition/` - 竞赛活动

## 模块设计原则

1. 每个模块独立封装，低耦合
2. 统一的接口规范
3. 支持热更新
4. 数据持久化
